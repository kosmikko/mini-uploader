# mini-uploader

Yet another JS upload plugin. Aimed to be minimalistic and easily extensible.

## Setup

    npm install

## Running tests

### Command line

Requires [Phantomjs](http://phantomjs.org/)

    make test

### Run webserver on project root

  e.g. 
  
    python -m SimpleHTTPServer 8001
    
  open browser in 
  
    http://localhost:8001/test/
